# Java-Number-Guessing-Game

A simple number guessing game crated using Java!

## Instructions for Build and Use

Steps to build and/or run the software:

1. Have JDK25 downloaded (https://www.oracle.com/java/technologies/downloads/#jdk25-windows)
2. To use in VS Code, download the extension pack (https://code.visualstudio.com/docs/java/java-tutorial)

Instructions for using the software:

1. Download numberGuessingGame folder
2. Run the Main.java file
3. Follow prompts and make guesses!

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* JDK25 (https://www.oracle.com/java/technologies/downloads/#jdk25-windows)
* VS Code
* VS Code Java extensions (https://code.visualstudio.com/docs/java/java-tutorial)
